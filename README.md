# Tour Places

Responsive Tour Places project made using React

<hr />

## What I Used

- React Bootstrap
- Bootstrap

### Installation Options

```
$ git clone https://github.com/HilalVildan/Tour-Places.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Tour-Places.git
```

```
$ yarn
```

```
$ yarn start
```
